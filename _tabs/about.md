# About HeyFinOps

Cloud FinOps에 대한 실무 경험과 인사이트를 공유하는 기술 블로그입니다.

## 다루는 주제

- **AWS Cost Optimization** — Reserved Instances, Savings Plans, Spot 전략
- **Azure Cost Management** — Advisor 활용, 리소스 태깅 전략
- **GCP Billing** — Committed Use Discounts, 빌링 분석
- **FinOps Framework** — Inform, Optimize, Operate 단계별 실무
- **Tooling** — 비용 시각화, 알림, 자동화 도구
